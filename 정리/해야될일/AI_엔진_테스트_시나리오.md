# AI 엔진 테스트 시나리오

## 테스트 목적
대화형 AI가 채팅 내용을 분석하여 일정을 정확하게 추출하는지 검증

---

## 시나리오 1: 명확한 날짜 + 시간 합의 ✅

**대화:**
```
철수: 다음주 화요일 2시에 만날까요?
영희: 좋아요!
민수: 저도 괜찮습니다
```

**예상 결과:**
```json
{
  "agreed": true,
  "summary": "미팅",
  "date": "2026-01-14",
  "startTime": "14:00",
  "endTime": "15:00",
  "location": null
}
```

---

## 테스트 방법

1. 테스트 채팅방 생성
2. 각 시나리오별로 메시지 입력
3. AI 제안이 나타나는지 확인
4. 제안 내용이 예상 결과와 일치하는지 검증
5. 오류 발생 시 로그 확인 및 프롬프트 개선

---

## 성능 지표

- **정확도**: 10개 시나리오 중 성공한 시나리오 수
- **응답 시간**: AI 분석 ~ 제안 표시까지 걸린 시간
- **False Positive**: 합의되지 않았는데 제안이 나타나는 경우
- **False Negative**: 합의되었는데 제안이 나타나지 않는 경우
